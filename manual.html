Overview

This is a website designed using HTML, CSS and JavaScript. Based on the website, a user is expected to select a birth date and their respective gender and an Akan name will be generated based on his/her inputs and Gender. The names will be as follows:

Male Names

  • Sunday: Kwasi
  • Monday: Kwadwo
  • Tuesday: Kwabena
  • Wednesday: Kwaku
  • Thursday: Yaw
  • Friday: Kofi
  • Saturday: Kwame

  • Monday: Adwoa
  • Tuesday: Abenna
  • Wednesday: Akwa
  • Thursday: Yaa
  • Friday: Afua
  • Saturday: Ama
